# KTTRACKER 📍

KTTRACKER is an Android location tracking application built using **Kotlin** and **Jetpack Compose**. The app tracks user location, stores it locally, and displays tracking information for monitoring and analysis.

---

# ✨ Features

* 📍 Real‑time location tracking
* 🗺️ Map integration (OpenStreetMap / Google Maps intent support)
* 💾 Local storage using Room Database
* 🔄 Background tracking using WorkManager / Foreground Service
* 🕒 Timestamp‑based location history
* 📊 Display tracked locations in a structured list
* 🔐 Runtime location permission handling

---

# 🏗️ Tech Stack

**Language:** Kotlin
**UI:** Jetpack Compose
**Architecture:** MVVM (Model‑View‑ViewModel)
**Database:** Room Database
**Background Tasks:** WorkManager / Foreground Service
**Location Services:** Fused Location Provider
**Build System:** Gradle (Kotlin DSL)

# 🚀 Installation

## 1. Clone the repository

```bash
git clone https://github.com/ApkDeveloper02/KTTRACKER.git
```

## 2. Open in Android Studio

* Open Android Studio
* Click **Open Project**
* Select the cloned folder

## 3. Sync Gradle

Wait for Gradle sync to finish.

## 4. Run the app

Click **Run ▶** and install on device/emulator.

---

# 🔐 Permissions Used

The app requires the following permissions:

```
ACCESS_FINE_LOCATION
ACCESS_COARSE_LOCATION
ACCESS_BACKGROUND_LOCATION
FOREGROUND_SERVICE
```
